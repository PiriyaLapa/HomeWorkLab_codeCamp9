grid และ flex แตกต่างกันอย่างไร?

Property Differection
Display:flex Display : flex จะเป็นการจัด element เรียงต่อกันและรองรับแบบ Respondsive element ต่างๆ ขยับไปตามขนาดของ หน้าจอที่เปลี่ยนไป
Display:grid เราสามารถจัด layout โดยการ row และ column ตามที่เราต้องการได้เลย ไม่ว่า element จะอยู่ซ้าย ขวา บน ล่าง โดยที่เราไม่ต้องใช้ float และสามารถ กำหนด element ให้อยู่ใน ตำแหน่งที่ต้องการได้อย่างอิสระ แต่ Display-grid นั้นไม่ออกถูกออกแบบเพื่อให้มาใช้งานแทน flex แต่มาเสริมให้ flex ทำงานที่ดีขึ้น

justify and align แตกต่างกันอย่างไร?

Property Differention
justify จะเป็นกันจัด element ไปตามแนวนอน จากว่าไปซ้าย
align เป็นการจัด element เป็นแนวตรงจาก บนลงล่าง
