Postion คืออะไร และ มี อะไรบ้าง?

position คืออะไร มีออะไรบ้าง
Position คือะไร property หนึ่งของ CSS เอาไว้กำหนดประเภทของการวางตำแหน่งของ element ต่างๆ บน layout ของเรา
  • Static
  • Relative
  • Absolute
  • Fixed
  • Sticky

การใช้งาน Static

static 1
static 2

CSS HTML 
#boxStatic {
  position: static;
  display: flex;
  justify-content: center;
  align-items: center;
  color: #ffff;
  border: 1px solid #4f5e5d;
  height: 60px;
  width: 60px;
  background-color: #f3827c;
}
	    
<div id="boxStatic">static 1</div>
<div id="boxStatic">static 2</div>

ปกติแล้ว element default ที่ไม่ได้มีการกำหนดค่า ให้กับ element ตัวPositon จะเป็น static อยู่แล้ว คือ มีจะ render ตามลำดับ document flow

การใช้งาน Sticky

position ตัวนี้จะมีความพิเศษซึ่งการทำงานของมันปกติจะเป็นแบบ relative แต่ถ้าเรา scroll จนมัน ตกจากจุด view point เมื่อไหร่มันจะทำตัวเป็น position fixed ทันที

z-index คืออะไรใช้อย่างไร

z-index คือ ใช้อย่างไร?
โดยปกติแล้วเราจะรู้จัก แกน x(นอน),y(ตั้ง) ซึ่งเราจะเห็นแค่สองมิติเท่านั้น แต่การที่เราจะเห็นภาพลึก หรือ ภาพที่ดูซ้อนกันนะเราต้องรู้จักแกน z-index โดยให้เราคิดเหมือนว่าวัตถุที่อยู่บนเส้น z-index จะเดินหน้าเข้ามาหาเรายิ่งเข้าใกล้เรามากเท่าไหร่ค่าจะเป็นบวกเท่านั้น ซึ่งกันสามารถประยุกต์ใช้กับงานซ้อน element ได้
