Inline Style

<p style=”font-size: 0.9em; width: 200px; border: solid 1px #000000”>ตัวหนังสือสีดำ อยู่ใน แทก </p>

Internal Style

การเขียนแบบ Internal (embedded) คือเขียนแยก CSS ออกจาก tag [X]HTML แต่อยู่ในหน้าเดียวกันกับไฟล์ [X]HTML โดยแทรก css เอาไว้ที่ <head></head> เช่น

External Style

คือการเขียน แยก css ออกมาเป็นไฟล์ .css ต่างหาก แล้วนำเข้าไปใช้ใน หน้าเว็บด้วย การเรียกโดย ใช้ โดยการพิมพ์ <link rel="stylesheet" href="externalcss.css" type="text/css" /> ไว้ที่ส่วน <head></head>ของหน้าเว็บ


For example CSS Selector

1. CLASS SELECTOR

            <style>
              .myclass{
                font: bold 12px Tahoma;color: #009;
              }
            </style>

วิธีการใช้งาน

            <p class='myclass'>Test to changeColor</p>

1. ID SELECTOR

            <style>
              .myclass{
                font: bold 12px Tahoma;color: #009;
              }
            </style>

วิธีการใช้งาน

            <p class='myclass'>Test to changeColor</p>

For example CSS Combinator

ตัวอย่างการใช้งาน comainator Descendant Select จะเป็นการเลือก element ที่อยู่ด้านในทั้งหมด ตัวอย่างการใช้งาน 

            <style>
              div p {
                background-color: skyblue;
              }
            </style>
            --------------------------------------------------
            <article>
  <h1>Descendant Selector</h1>
  <div>
    <p>Descendant Selector 1</p>
    <p>Descendant Selector 2</p>
    <section>
      <p>Descendant Selector 3</p>
      <p>Descendant Selector 4</p>
    </section>
  </div>
  <p>Descendant Selector 5</p>
  <p>Descendant Selector 6</p>

Property Values Unit
คุณสมบัติของ css เช่น width, height, position คือค่าของ property นั้นๆ ครับ เช่น 500px สำหรับความกว้าง

    Absolute

  • cm (centimeters)
  • mm (millimeters)
  • in (inches)
  • px (pixels)
  • pt (points)
  • pc (picas

    Relative

  • em (Relative to the font size
  • ex (Relative to x-height of the current font
  • ch (Relative to width of the 0
  • rem (Relative to font-size of the root element
  • vw (Relative 1% on width of the viewport
  • vh (Relative 1% on height of the hight viewport
  • vmin (Relative1% of viewport * small dimersion
  • vmax (Relative to 1% viewport * large dimersion
  • % (Relative to the parent element
